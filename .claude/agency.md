# AI-Assisted Development Learning Agency (MCP-based)

## Development Environment Rules

### DO NOT run these commands (already running with hot-reload):
- `npm run dev` in frontend/
- `npm run dev` in backend-node/

### If you need to restart dev servers:
```bash
./start-dev.sh
```

---

This document is a ready-to-use reference guide for using AI to learn software development. It includes:

*   ✅ A single agency file
*   ✅ Markdown format
*   ✅ Designed for absolute beginners (who cannot code at all)
*   ✅ Adaptable for any programming language or tech stack
*   ✅ Built-in rules to prevent over-reliance on AI

---

## How to Use This Document (Important)

### 1️⃣ When Starting a New Project

*   Keep this `.md` file in the project root.
*   Always tell the AI assistant:
    ```
    Follow the rules in this agency file strictly.
    ```

### 2️⃣ Learning Flow

*   Follow the project-specific learning steps in sequential order.
*   **Never skip steps.** (e.g., Do not attempt complex optimizations before implementing core features ❌).

### 3️⃣ When You Get Stuck

*   Do not just ask for the complete code.
*   Instead, copy and paste the relevant prompt from within this agency file to ask your question. This helps you learn the right way to ask questions and get guidance.
